AXIOM CODE REVIEWER — INTERVIEW QUESTION KNOWLEDGE BASE
========================================================
This knowledge base covers the Blind 75 question set.
For questions outside this list, use model knowledge to evaluate.

---

QUESTION: Two Sum
DIFFICULTY: Easy
TOPIC: Arrays
OPTIMAL APPROACH: Hash Map
TIME COMPLEXITY: O(n)
SPACE COMPLEXITY: O(n)
DESCRIPTION: Given an array of integers and a target integer, return the indices of the two numbers that add up to the target. Each input has exactly one solution and you may not use the same element twice.
EDGE CASES: empty array, single element, negative numbers, duplicate values, target is zero, no valid pair exists
COMMON MISTAKES: Using nested loops (O(n²)), not storing indices correctly, modifying the array

QUESTION: Best Time to Buy and Sell Stock
DIFFICULTY: Easy
TOPIC: Arrays
OPTIMAL APPROACH: Single Pass / Sliding Window
TIME COMPLEXITY: O(n)
SPACE COMPLEXITY: O(1)
DESCRIPTION: Given an array of prices where prices[i] is the price on day i, find the maximum profit by choosing one day to buy and a later day to sell. Return 0 if no profit is possible.
EDGE CASES: all prices decreasing, single element, all same prices, two elements
COMMON MISTAKES: Not tracking minimum price correctly, allowing sell before buy
